# Fire Zoom

- training app in React Native / Typescript along with Socket IO

# Scripts

Install dependencies in both (frontend/backend)

```
  yarn install
```

In frontend folder to run app
```
  yarn start
```

and choose ios/android device

In backend folder to run server
```
  node index.js
```
